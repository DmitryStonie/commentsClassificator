Данная программа состоит из нескольких файлов:
main.py - главный файл, с которого начинается исполнение кода (именно его нужно запускать). 

classificator.py содержит, функцию train, которая вызывается в main. 
В качестве аргумента в нее передается другая функция этого файла, представляющая тренируемую модель.
Одна из шести функций:
naive_bayes
neighbors
SGD
linear_SVC
decision_tree
MLP
Также функция train выводит результаты полученного обучения. Результаты оцениваются на тестовой выборке.


tsv-parser.py содержит функцию для предварительной обработки (векторизации) данных. 
Т.е функция, которая делает из текстовых комментариев вектора чисел.

places.py - файл содержит набор ссылок на вкладки с отзывами о местах. Ключевым моментом является то, что
это именно вкладка отзывы, а не просто карточка места. Для выполнения работы использовался словарь theaters
(который строится исходя из списка places), хотя в файле присутствуют и другие ссылки. Они не были использованы.

comments-downloader.py - в этом файле содержатся функции, осуществляющие парсинг комментариев. Для работы с
драйвером браузера google chrome использовалась библиотека Selenium.

def scroll(driver, limit): скроллит страничку в низ с учетом числа необходимых комментариев(limit).
def chooseType(driver, type): выбирает сортировку комментариев: по возрастанию рейтинга либо по убыванию
def openComments(driver): жмет на кнопки "Ещё", чтобы открыть полный текст всех комментариев на странице
def get_comments(url, driver, comments_to_parse, type): сохраняет текст комментария в массив results
def write_results(results, filename): записывает массив комментариев в файл filename.tsv
def get_samples(num_of_comments, train_ratio): парсит num_of_comments комментариев, из которых половина
положительных, половина отрицательных. Записывает их в файлы train_sample.tsv и test_sample.tsv, где
train_ratio * num_of_comments комментариев попадает в train_sample.tsv, оставшиеся в test_sample.tsv.

Выборки содержатся в файлах test_sample.tsv и train_sample.tsv.
Формат tsv представляет из себя набор строк, где первая строка - название столбцов.
Далее идут строки с информацией в виде: комментарий знак табуляции класс принадлежности.

Вывод программы - таблица-пример из десяти комментариев и результатов классификации, 
точность прогнозирования, матрица ошибок.

Для запуска программы возможно придется установить некоторые из использованных библиотек.
